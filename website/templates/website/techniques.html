Layouts Made use of CSS Grid and CSS Flexbox to build the pages' layouts (PC, Tablet and Smartphone) Use of Grids and Flexbox can be verified in all pages (e.g: header, main, nav and footer elements), with the Home page having a different layout from other pages.

Templates Use of Django templates for every page of the website, having base HTML files that were extended and added upon by specific pages' HTML files, making use of Django's template language to make use of variables passed from the View into the HTML. Use of Django Templates is verified in every page of the website.
Python Use of the Python programming language with Django to create the Models and Views that were used in the project. Use of Python is verified in every page of the website.
